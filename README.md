## Informações Básicas 

- Estrutura de um programa em Pascal:
```pascal
program exemplo;
var variavel_exemplo;

begin
  (* comando 1; *)
  (* comando 2; *)
  (* comando 3; *)
end.
```

- Como compilar:
```
  fpc exemplo.pas
```

- Como Executar:
```
  ./exemplo
```
